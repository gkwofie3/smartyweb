Quickstart Guide

To get started with a new Smarty installation, follow these three steps:

  1. Download the Smarty Server Bundle from your dashboard.
  2. Execute smarty-probe --hwid to get your unique machine ID.
  3. Paste the ID into the Activation Portal and download your smarty.lic file.
Make sure your local firewall allows outbound traffic to api.smartyportal.com on port 443 for license validation.

Hardware Identity (HWID)

Smarty uses a SHA-256 fingerprint of the following components:

  • Motherboard UUID
  • CPU ID (First Processor)
  • Primary Disk Serial Number

If you replace more than one of these components, the activation will be voided. Standard and Pro users can self-transfer licenses once per month.

Broker API Configuration

The Broker API allows your mobile device to route through our secure proxy to reach your local network. Configure your settings.json on the local server:

{
  "system_id": "london-plant-01",
  "broker_url": "https://api.smartyportal.com/v1/",
  "local_port": 8443,
  "use_proxy": true
}
